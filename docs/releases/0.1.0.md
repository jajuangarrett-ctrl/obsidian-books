# Obsidian Books 0.1.0

Obsidian Books is a public development preview of an Apple Books-inspired
Markdown reader for Obsidian, built from the proven pagination foundation of
MD Reader.

## Highlights

- Responsive single-page and two-page reading with None, Horizontal Slide, and
  3D Page Turn transitions.
- Searchable bookshelf for standalone notes and folder books defined by
  `Book.md`, with covers, metadata, ordered chapters, progress, and resume.
- Theme-aware white, cream, sepia, and dark surfaces plus adjustable typography,
  spacing, margins, and page geometry.
- Keyboard, mouse-wheel, tap-zone, swipe, visible-control, and chapter
  navigation with immersive reading, live progress, and reading-time estimates.
- Persistent bookmarks, source-safe highlights, and quote capture to shared,
  per-book, or configured annotation destinations.
- Resilient text anchors and `obsidian://books-open` links that reopen saved
  books, chapters, and reading locations.
- Native vertical fallback for PDFs and oversized or highly interactive content,
  including restorable scroll progress.
- Legacy MD Reader settings and position migration, MIT attribution, and only
  supported Obsidian APIs.

## Validation

- Formatting, ESLint, strict TypeScript, 25 unit tests, and the production build
  pass locally and in GitHub Actions.
- Directly validated in Obsidian 1.12.7 using a dedicated test vault.
- Compatibility passes cover prose, headings, images, callouts, tables, inline
  and fenced code, Dataview 0.5.68, Tasks 8.2.2, Mermaid, note and block embeds,
  PDFs, audio, video, footnotes, links, checkboxes, and oversized content.
- Responsive viewport, synthetic touch, reduced-motion, and increased-contrast
  emulation passes are recorded in the test checklist.
- Release assets were built on GitHub, SHA-256 digests were checked after
  download, and the provenance attestation was verified.

Physical iPad touch, orientation, and VoiceOver testing remains the one
device-only validation item.

## Install

Create `.obsidian/plugins/obsidian-books/` in a test vault, place `main.js`,
`manifest.json`, and `styles.css` from this release in that folder, restart
Obsidian, and enable **Obsidian Books** under Community plugins.

See the repository README for folder-book syntax, annotation behavior, complete
documentation, and known limitations.
